¿CÓMO FUNCIONA TROPBOT?

En el diagrama se detalla como se conforma la plataforma móvil y como es el enlace del sistema de adquisición con la interfaz gráfica para el usuario. Esta plataforma consta de una etapa de adquisición de las variables físicas a través de los sensores y módulos con salida digital por I2C, la siguiente etapa consiste en acondicionar los datos tomados, seguido, esta el sistema embedido de la Raspberry Pi que describe las operaciones que se realizan de procesamiento y por último, en este sistema se muestra el sistema de comunicación para enviar los datos a la nube y el control de los motores.

Por medio de la tarjeta Raspberry PI, se establece comunicación directamente a la interfaz de usuario, para enviar estos datos y visualizarlos en la nube.
